# TikTok Ads — Лучшие практики

## Кампании App Promotion

### Структура
- Всегда ручная кампания (REGULAR_CAMPAIGN), не Smart+
- Бюджет на адсетах (ABO), не на кампании (CBO)
- Всегда 5 адсетов на кампанию — TikTok алгоритм лучше обучается при нескольких адсетах
- Старт кампании: 6:01 МСК следующих суток — алгоритм получает полные сутки для обучения

### iOS 14+
- Всегда включать специализированную кампанию iOS 14 для iOS-трафика
- Отключать SKAN атрибуцию — используем собственный трекинг (Adjust/AppsFlyer)
- Отключать страницу профиля приложения — избегаем лишних кликов, ведём на прямой трекер

### Android
- Поле "Приложение" заполнять на уровне адсета
- iOS 14 настройки не применяются

### Оптимизация
- Стандарт: In-App Event → PURCHASE
- Если PURCHASE < 50 событий/7 дней → спросить пользователя (REGISTRATION или INSTALL)
- Минимум 50 событий/неделю для стабильного обучения алгоритма

### Плейсмент
- ТОЛЬКО TikTok (PLACEMENT_TIKTOK)
- Pangle и другие сети не используем

## Адсеты

### Таргетинг
- Гео: запрашивать перед заливом — зависит от оффера
- Возраст: 18–55+ (мужчины и женщины) — широкий таргетинг для обучения алгоритма
- Интересы: не трогать — широкий таргетинг лучше для in-app event оптимизации

### Бюджет и ставки
- Всегда автоставка (BID_TYPE_NO_BID) — TikTok сам оптимизирует
- **Cost Cap не использовать — только по прямому указанию пользователя**
- Минимальный бюджет: $20/день на адсет — ниже = плохое обучение

### Суб-параметры
- Каждый адсет имеет уникальный суб (avfrites1...5) — обязательно для трекинга конверсий
- Имя адсета = суб-значение — позволяет сразу видеть маппинг в отчётах партнёрской сети

## Объявления

### Умные креативы (Smart Creative)
- Всегда включать — TikTok автоматически тестирует варианты текста и CTA
- Загружать минимум 3–5 видео на кампанию для A/B теста

### Видео-креативы
- Переименовывать файлы перед загрузкой — избегать конфликтов имён между кампаниями
- Формат: вертикальное видео 9:16, 720p+, 15–30 секунд оптимально
- Первые 3 секунды критичны — должны сразу захватывать внимание

### Профиль
- Использовать профиль соответствующий приложению или оферу
- Spark Ads — выключать всегда (используем кастомные объявления)

### Тексты
- "Play Now" / "Start Play" — стандартные тексты
- Коротко, с призывом к действию

### CTA кнопки
- Android: Download + Install + Google Play (2–3 кнопки)
- iOS: Download + Install + App Store (2–3 кнопки)

## Трекинг

### AppsFlyer
- Не требует токен при подключении через TikTok Event Manager
- События: выбирать "All events" при подключении

### Adjust
- Требует App Token из Adjust Dashboard
- События: выбирать "All events" при подключении
- Ждать 5–10 минут после настройки до появления в `tt_list_apps`

### Трекинг App кампаний
- Трекинговая ссылка для App **не нужна** — трекинг идёт автоматически через MMP
- Приложение подключается к TikTok через Event Manager (AppsFlyer / Adjust)
- Атрибуция конверсий на стороне партнёрской сети идёт по нейминг кампании: суб-параметр `sub7=avfrites` в имени кампании = уникальный идентификатор

### Трекинг Website кампаний
- Трекинговая ссылка передаётся в `click_tracking_url` на уровне адсета
- Формат: `?{SUB_PARAM}={SUB_VALUE}&clickid={CLICKID}&...`

## Типичные ошибки

| Ошибка | Причина | Решение |
|--------|---------|---------|
| TikTok placement недоступен | Кабинет не верифицирован | Сообщить пользователю, не заливать |
| PURCHASE event недоступен | Трекинг не настроен | Проверить Adjust/AppsFlyer, спросить альтернативу |
| Кампания не выходит из обучения | Мало данных / низкий бюджет | Увеличить бюджет, расширить гео |
| Высокий CPI сразу после запуска | Нормально — алгоритм обучается | Не трогать первые 3–7 дней |
| Видео не загружается | Конфликт имён файлов | Переименовать перед загрузкой |
