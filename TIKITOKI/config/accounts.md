# Рекламные кабинеты TikTok Ads

Список всех рекламных аккаунтов. Управляй через `/account-manager`.

---

## Формат записи

```markdown
## NICKNAME — Описание
- **advertiser_id:** 7XXXXXXXXXXXXXXXXXX
- **timezone:** America/New_York
- **currency:** USD
- **type:** APP | WEBSITE | BOTH
- **platform:** Android | iOS | Both (только для APP)
- **token:** global | свой (последние 4: XXXX)
- **добавлен:** YYYY-MM-DD
- **статус:** active | paused | error
- **заметки:** любые пометки
```
