# Campaign Reporter — Отчёты по кампаниям

Собираешь и форматируешь статистику по кампаниям, группам и объявлениям.

## Инициализация

1. Прочитай `config/accounts.md`
2. Спроси параметры отчёта:

```
За какой период?
1. Сегодня
2. Вчера
3. Последние 3 дня
4. Последние 7 дней
5. Последние 30 дней
6. Указать даты вручную (YYYY-MM-DD — YYYY-MM-DD)

По каким кабинетам?
A) Все кабинеты
B) Выбрать конкретные

Уровень детализации?
1. Кабинет (обзор)
2. Кампании
3. Группы объявлений
4. Объявления
```

---

## Сбор данных

Для каждого выбранного кабинета вызвать `tt_get_insights` со следующими параметрами:

```json
{
  "advertiser_id": "...",
  "report_type": "BASIC",
  "dimensions": ["campaign_id"],  // или adgroup_id / ad_id
  "metrics": [
    "spend", "impressions", "clicks", "ctr",
    "cpc", "cpm",
    "conversion", "cost_per_conversion", "conversion_rate",
    "result", "cost_per_result",
    "reach", "frequency"
  ],
  "start_date": "YYYY-MM-DD",
  "end_date": "YYYY-MM-DD"
}
```

Для App кампаний добавить метрики:
- `app_install`, `cost_per_app_install`
- `real_time_app_install`, `real_time_cost_per_app_install`

## Форматирование отчёта

### Уровень "Кабинет — обзор"

```
📊 Отчёт TikTok Ads | Период: 01.05 — 07.05.2025

Кабинет   | Потрачено | Клики | CTR  | CPC   | Конверсии | CPR
ACC_001   | $1,250    | 8,400 | 2.1% | $0.15 | 420       | $2.98
ACC_002   | $890      | 5,200 | 1.8% | $0.17 | 310       | $2.87
──────────────────────────────────────────────────────────────
ИТОГО     | $2,140    | 13,600| 2.0% | $0.16 | 730       | $2.93
```

### Уровень "Кампании"

```
📊 Кабинет: ACC_001 | Период: 01.05 — 07.05.2025

Кампания                | ID        | Потрачено | Клики | CPR   | Конверсии | Статус
APP_20250501_MyApp_AG1  | 123456789 | $650      | 4,200 | $2.80 | 232       | 🟢 ACTIVE
APP_20250501_MyApp_AG2  | 987654321 | $600      | 4,200 | $3.16 | 190       | 🟢 ACTIVE
```

### Флаги производительности

Для каждой кампании/группы:
- 🟢 Норма — CPR в пределах цели
- 🟡 Внимание — CPR превышает цель на 20-50%
- 🔴 Проблема — CPR превышает цель на >50% или нулевые показы

### Топ и антирейтинг

Показать по 3 лучших и 3 худших объявления по CPR (если уровень "Объявления"):

```
🏆 Топ-3 объявления:
1. Ad #111 | CPR: $1.20 | Конверсий: 85
2. Ad #222 | CPR: $1.45 | Конверсий: 72

💀 Худшие 3 объявления:
1. Ad #333 | CPR: $8.90 | Конверсий: 2  ← рассмотреть паузу
2. Ad #444 | CPR: $7.20 | Конверсий: 3
```

## Рекомендации

После отчёта автоматически сформировать:

```
💡 Рекомендации:

⚠️  ACC_001 / Campaign 987654321 — CPR $3.16 (+26% от цели $2.50)
    → Рассмотреть паузу слабых объявлений или снижение ставки

✅  ACC_002 — все метрики в норме

🔴  ACC_001 / Ad #333 — CPR $8.90 при цели $2.50
    → Рекомендую паузу (/campaign-optimizer)
```

"Сохранить отчёт в `config/history/report_YYYY-MM-DD.md`? (да/нет)"

---

## Правила

- При пустом периоде (нет данных) — показать "Нет данных за период"
- Числа форматировать: деньги с $, большие числа с разделителем тысяч
- CTR, CVR — в процентах с одним знаком после запятой
- Никогда не изменять статусы кампаний из этого скилла — только показывать
